MessageBar

An Android Toast replacement, similar to the one seen in the GMail app. Multiple messages can be posted in succession and will be retained on screen rotation

DEMO can be download at Google Play

main customizable functionality

  1. can be added to every view and adjusts the parent automatically to be capable to display a non layout relevant message bar (floating view)
  2. display duration
  3. disposing mode (automatic, button click)
  4. without button or with one or two buttons
  5. display remaining display time
  6. customizable message bar style
  7. extended messages for convenience
